# Shopify Inc. (SHOP.TO) — DCF Valuation Model

## Overview
A discounted cash flow valuation model for Shopify Inc. built in Excel as part of self-directed finance training (FMVA + AVFM).

## Methodology
- 5-year free cash flow projection (2025–2029)
- WACC-based discounting using CAPM
- Terminal value using Gordon Growth Model
- Sensitivity analysis across WACC and terminal growth rate

## Key Assumptions
- Revenue growth: 22% → 12% (declining over projection period)
- Gross margin: ~50% (consistent with FY2024 actuals)
- WACC: 10.0%
- Terminal growth rate: 3.0%

## Data Sources
- Shopify Annual Reports 2022–2024 (investor relations)
- Damodaran equity risk premium (Jan 2026)
- Canada 10-year government bond yield

## Model Structure
| Tab | Contents |
|-----|----------|
| Cover | Model overview and navigation |
| Assumptions | All inputs — growth, margins, WACC |
| Income Statement | Historical 2022–2024 and projected 2025–2029 |
| Cash Flow | Free cash flow bridge |
| DCF Output | Valuation, implied share price, sensitivity table |
